Hardcore Viking

Descrição

Um jogo muuito legal, vencedor de varios premios, como o Luan Gameplay Awards 2017, o maior troféu da américa hispânica

Competências
  • code Programação
  • create Design
Imagens
class
Metodologia

O trabalho se dedenvolveu de forma que os professores auxiliares de sua execução faziam checkpoints frequentes semanais, dessa forma era decidido o formato que o programa em Python tomaria, no casod e um jogo em plataforma. Esses checkpoints eram cada vez mais desafiadores, além de que todos os trabalhos dos grupos eram diferentes e personalizados aos interesses dos integrantes.

build
Técnicas Aplicadas

O grupo utilizou de uma biblioteca do Python 3.6 conhecida como Pygame, na qual é possível ter uma base para criação de janelas e objetos em tela. A partir disso é possível criar o ambiente gráfico, que é editado por funções rodando em um grande loop(Game Loop), de forma que podem ser abrangidos fórmulas de física, movimentação com o teclado, randomização, além de outras particulariedades de jogos atuais.

grade
Recepção

O jogo foi muito bem recebido pelos professores, de forma que atingiu nota máxima e no semestre seguintes, foi exibido no Momento Insper, evento da faculdade com exposição de trabalhos para pretendentes estudantes do Insper.
